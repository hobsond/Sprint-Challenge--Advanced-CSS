
1. What is the difference between an adaptive website and a fully responsive website?
An adaptive website will be based on percentages, an adaptive website will adjust to its size, a responsive website reacts to the change of the screen sizes.

2. Describe what it means to be mobile first vs desktop first.
Mobile first is designing your website for mobile display and min-width media query to scale up.

3. What does `font-size: 62.5%` in the `html` tag do for us when using `rem` units?
The default browser is set to 16px so if you make the root font size to be 62.5% that makes it equal to 10px which makes it easier to convert rem.
